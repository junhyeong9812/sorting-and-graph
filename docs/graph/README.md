# 그래프 탐색 (Graph Traversal)

## 개요

그래프의 모든 정점(노드)을 체계적으로 방문하는 방법입니다.

## 알고리즘 목록

|---------|------|---------|---------|

| BFS | [bfs.md](./bfs.md) | 큐 | O |

## 그래프 기초

### 그래프란?
정점(Vertex)과 간선(Edge)으로 이루어진 자료구조입니다.
```
    A ---- B
    |      |
    |      |
    C ---- D ---- E
```

### 그래프 종류

| 종류 | 설명 |
|-----|------|
| 무방향 그래프 | 간선에 방향 없음 (A-B = B-A) |
| 방향 그래프 | 간선에 방향 있음 (A→B ≠ B→A) |
| 가중치 그래프 | 간선에 비용/거리 있음 |

### 그래프 표현 방식

**인접 행렬 (Adjacency Matrix)**
```
    A  B  C  D
A [ 0, 1, 1, 0 ]
B [ 1, 0, 0, 1 ]
C [ 1, 0, 0, 1 ]
D [ 0, 1, 1, 0 ]

공간: O(V²)
간선 확인: O(1)
```

**인접 리스트 (Adjacency List)**
```
A: [B, C]
B: [A, D]
C: [A, D]
D: [B, C]

공간: O(V + E)
간선 확인: O(degree)
```

## DFS vs BFS 비교

| 특성 | DFS | BFS |
|-----|-----|-----|
| 자료구조 | 스택/재귀 | 큐 |
| 탐색 방식 | 깊이 우선 | 너비 우선 |
| 메모리 | 경로 길이 비례 | 너비 비례 |
| 최단 경로 | 보장 안됨 | **보장됨** |

### 언제 DFS?
- 모든 경로 탐색
- 백트래킹 (N-Queen, 미로)
- 사이클 탐지
- 위상 정렬

### 언제 BFS?
- **최단 경로** (가중치 없는 그래프)
- 레벨별 탐색
- 최소 이동 횟수
